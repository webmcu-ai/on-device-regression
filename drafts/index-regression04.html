XIAO ML Kit Distance Regression Training over WebSerial

Train a two-headed CNN (distance + confidence) on the XIAO ESP32-S3 Sense camera to predict how far an object is from the lens — with post-training least-squares calibration for extrapolation beyond the training range.

Base github: github.com/webmcu-ai/webmcu-regression-web  ·  On-device github: github.com/webmcu-ai/on-device-regression
Reference vision web page (torchjs145): torchjs145.html  ·  Reference firmware (v78): firmware.ino

arXiv Paper 1 (on-device): arxiv.org/abs/2604.23012  ·  arXiv Paper 2 (WebSerial): arxiv.org/abs/2604.22834



1 · Flash Firmware

Flash compiled ESP32 .bin files directly from the browser using esptool-js v0.5.7.
Requires Chrome/Edge · HTTPS or localhost · USB CDC On Boot: Enabled

Source code: firmware-regression.ino

⚠ MUST REFRESH WEBPAGE FIRST before flashing, then unplug/replug after flash completes.

▸ Option A — Sketch Only (flash.ino.bin)

Flashes only the application partition at 0x10000. Fast. Preserves bootloader and NVS.

Browse (optional)
Default: flash.ino.bin · Browse to override

▸ Option B — Full Merged Flash (flash.ino.merged.bin)

Flashes bootloader + partition table + app at 0x0. Use for fresh chips.

Browse (optional)
Default: flash.ino.merged.bin · Browse to override
