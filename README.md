# on-device-regression
on device regression using the seeedstudio xiao esp32s3 sense in the form of the xiao ml kit


Started May 14, 2026


Needs the $22-38.90 USD [XIAOML kit](https://www.seeedstudio.com/The-XIAOML-Kit.html) for full on-device deployment.

Serial monitoring works with just the $14.90 USD [XIAO ESP32S3 Sense](https://www.seeedstudio.com/Seeed-Studio-XIAO-ESP32S3-Sense-Pre-Soldered-p-6335.html)


Upload code, train: blank, 2, 5, 10 whatever units from an object for units use cm or inches etc. Train multiple times then infer. Extrapolation now works
